# 🍽️ Lyte Food — Dashboard de Gestion SMS

**Lyte Food** est une solution moderne de gestion pour restaurants qui transforme les messages clients (SMS) en données exploitables directement sur un tableau de bord.

### ✨ Fonctionnalités :
*   **Tableau de bord en temps réel :** Statistiques sur les réservations, les commandes et les SMS reçus.
*   **Gestion des Tables :** Visualisation rapide de l'état des tables (Libre, Réservée, Occupée).
*   **Suivi des Commandes :** Gestion du flux des commandes de la préparation à la livraison.
*   **Simulateur SMS :** Outil intégré pour tester les réponses automatiques du système.
*   **Design Premium :** Interface épurée utilisant les polices *Syne* et *DM Sans* pour une expérience utilisateur optimale.

### 🚀 Technologies utilisées :
*   **Frontend :** HTML5, CSS3 (Variables CSS, Grid, Flexbox)
*   **Logic :** JavaScript (ES6+)
*   **Backend :** Prêt pour intégration avec API REST
